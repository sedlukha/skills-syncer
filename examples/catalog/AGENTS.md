# Shared Project Instructions (example)

This block is merged into the top of every consuming repo's `AGENTS.md` by
skills-syncer, fenced with markers. Edit it **here, in the catalog** — not in the
consuming repos, where a re-sync would overwrite it. Whatever a repo writes below
the fenced block is preserved.

## Language

Write everything in plain, simple English — skills, comments, commits, PRs.

## House style

- Prefer the smallest change that solves the problem.
- Name things for what they do, not how they work.
- When a rule has a reason, state the reason.
